Graph Editing Tool For Power System

Introduction Description Example Training
  • In a power system, there are several kinds of elements: Bus, Generator, transmission line, transformer, regulator, compensator and so on.
  • The project classifies those elements as two categories: The elements which can consume or generate energy and the elements only for energy transmission. the first class converts as a node and the second class converts as an edge.
  • In the test, it uses the Nordic system, after converting into the node-edge graph, it contains 74 nodes and 102 edges.
  • Blackout, also called power outage, is a collapse of power system. It will lead to the damage of grid and the loss of energy.
  • Faults at power stations, electric transmission lines damage can lead to the voltage fluctuation and Insufficient reactive power. It may cause the system collapse.
  • In the project, the disturbance will create manually.
